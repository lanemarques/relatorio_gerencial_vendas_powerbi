# 📊 Relatório Gerencial de Vendas - Power BI

## 📌 Descrição

Projeto desenvolvido como parte do desafio da DIO, com o objetivo de criar um relatório gerencial de vendas utilizando o Power BI.

---

## 📊 Dados Utilizados

Base de dados **Sample Financials**, disponibilizada no repositório:

https://github.com/julianazanelatto/power_bi_analyst

---

## 📄 Estrutura do Relatório

O relatório foi dividido em 2 páginas:

* **Página 1:** Visão geral com indicadores e gráficos principais
* **Página 2:** Análise detalhada dos dados

---

## ⚙️ Recursos Utilizados

* Segmentadores (filtros)
* Botões de navegação
* Indicadores visuais
* Gráficos interativos

---

## 📎 Arquivos

* Relatório em **PDF** para visualização
* Arquivo **.pbix** para edição no Power BI

## 🛠️ Ferramenta
